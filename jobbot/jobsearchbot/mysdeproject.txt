PROJECT 1 — AI-Powered Learning Management System (Short + Detailed)
Built a scalable AI-enhanced LMS (MERN + Flask) to automate assessments and viva.
Tech: React, Node/Express, MongoDB, Flask, ChromaDB, Gemini 1.5 Flash, GOT-OCR, LMNT TTS, YOLOv8, Eden API.
Flow:
• Teachers upload chapter → converted into vector embeddings stored in ChromaDB (RAG).
• Students upload answer PDF → React → Node → OCR → plagiarism check → evaluated via Gemini with rubrics (logic, language, CT).
• Scores + feedback stored in MongoDB and shown on React dashboard.
AI Viva:
• Teacher provides 5-min audio → Flask clones voice with LMNT TTS.
• React Speech Recognition captures student responses.
• YOLOv8 performs proctoring (phone detection, multiple faces).
• Flask evaluates student answers using RAG + LLM and saves the viva report.
Why 2 Backends:
Node for auth, routing, dashboards Flask for Python AI ecosystem. Communication via REST APIs.
Extra Features: AI quiz generator, mind-maps, and 3D classroom visualizations.

PROJECT 2— JobHunt
AI-powered job search and resume optimization platform.
Tech: Spring Boot, React.js, Python (AI/ML), AWS, WebClient, Jsoup, LLMs
Functionality:
• Users upload resume + job description.
• AI tailors resume based on required skills in the JD.
• Generates personalized LinkedIn recruiter messages.
• Scrapes LinkedIn job listings based on filters (location, date).
• Ranks jobs using AI-based resume–JD matching.
• Auto-generates custom cover letters and job match score.
Backend & AI:
• Spring Boot REST APIs with WebClient for fetching LinkedIn pages.
• Jsoup used for extracting job data from HTML.
• Python scripts compute TF-IDF + cosine similarity for job matching.
• Dual-LLM pipeline generates and refines resumes and cover letters.
Purpose:
Automate job searching and resume customization while learning full-stack development, AI integration, web scraping, and real-world automation workflows.
